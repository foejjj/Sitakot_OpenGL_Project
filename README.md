# OpenGL Graphics Simulation (Sitakot Project)

## Overview
This project is a 2D interactive graphics simulation developed using C++ and OpenGL (GLUT). It demonstrates real-time rendering, animation, and user interaction.

## Features
- Day/Night mode toggle
- Weather effects (rain, thunder)
- Moving clouds and birds
- Interactive keyboard controls

## Technologies Used
- C++
- OpenGL (GLUT)

## Concepts Covered
- Transformation (translation, scaling, rotation)
- Rendering pipeline
- Coordinate systems
- Real-time animation
